# 🛒 Smart Purchase Reorder System

---

## 📌 Description

AI-based web application that automates inventory tracking and uses predictive analytics to forecast demand and generate intelligent purchase recommendations for efficient stock management.

---

## 🚀 Features

* 📦 Inventory tracking and management
* 📉 Low-stock detection
* 🔮 AI-based demand forecasting
* 🧾 Automated purchase order generation
* 📊 Dashboard for analytics and reports

---

## 🛠️ Tech Stack

* Python
* Django
* MySQL
* HTML, CSS
* Predictive Analytics

---

## ⚙️ Setup & Run

1. Clone the repository

```bash
git clone https://github.com/Swathy-Chandran/smart-purchase-reorder-system.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the server

```bash
python manage.py runserver
```
